# Maintainers

`bbsolver` is currently maintained by the author. Issues, pull requests,
and discussions are welcome on the public repository.

## Patch policy

- Public contributions are accepted via pull request.
- Core solver patches go through a normal review on the public branch.
- Tagged releases (`v1.0.x`, `v1.0.0`, etc.) follow semantic versioning;
  see `CHANGELOG.md` for the per-tag scope.

## Reporting issues

Use the GitHub issue tracker. For benchmark reproduction questions,
include the `request_id` from `benchmarks/corpus/corpus_manifest.csv`.

## Security

See `SECURITY.md`.
